commands to run the App:
To run frontend - {npm start}
To run backend - {node server.js} 
To run both simultaneously - {npm run dev} - {path -> {my-project}}
To run tailwind css - {npx tailwindcss -i ./src/styles/tailwind.css -o ./src/styles/output.css --watch (must kept running while performing)}.- {path -> {my-project-frontend}}
