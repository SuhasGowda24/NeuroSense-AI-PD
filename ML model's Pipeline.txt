2-Classification-Model Workflow:

The Image drawn in frontend(DrawingPad) calculates and shows the Table with -> Y, X Coordinate, TimeStamp, ButtonState, Pressure.

The Backend(Node.js/Express.js) - Image is stored in Cloudinary and the img url is then stored in MongoDB Atlas with the Table Attributes.

After saving Image(URL) + Attribute (points), The Backend(Node.js/Express.js) calls {(the HuggingFace) - Converts the table contents(raw JSON strokes) to ML features(trained handwriting features) and sends the ML required data to ML model to process}.and Then HuggingFace will process by predicting whether the person is PD or not.

HuggingFace prediction + confidence are sent to Backend(Node.js/Express.js).

Then BACKEND stores it in MongoDB.

Backend sends prediction back to frontend UI.
